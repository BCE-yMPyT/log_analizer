Программа предназначена для парсинга строк из логов nginx.

index.py - Вмещает в себя сценарий для обработки логов, с
	   использованием нижеследующих модулей.
  	   Пользователь выбирает необходимую операцию для лога,
	   и вводит соответствующую информацию для своего запроса.
	   Файл открывается как через командную строку, так и с
	   помощью редактора, поддерживающего выполнение модулей.

parcer.py - Содержит в себе необходимое Регулярное Выражение, и
	    функцию, которая принимает файл и возвращает данные
	    в более "удобном" формате для проведения последующих
	    операций т.е. список из словарей.

filters.py - Модуль-фильтр для выборки данных. Функция, принимает:
	     - файл данных .тхт
	     - начальную дату (включительно)
	     - конечную дату (не включительно,
  	       поэтому, если надо проверить 1 день,
  	       необходимо в качестве конечной даты
  	       ставить следующий).
	     И создаёт список данных логов, которые находятся
	     в этом временном промежутке.

redusers.py - Модуль для получения среднестатистических данных,
	      вмещающий функцию, которая принимает массив данных
	      и выводит количество запросов за временной промежуток
	      и среднее количество запросов.

dbwriter.py - Модуль для записи информации о логах в базу данных.

getter.py - Модуль для получения необходимой записи в таблицу БД.

archive_processor.py - Модуль для обработки архивов.
